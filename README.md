# TimeRewind

TimeRewind is an AI-powered Android application that helps users capture, organize, and retrieve memories through voice recordings. It combines speech transcription, semantic search, and emotion analysis to create an intelligent, privacy-focused memory assistant.

## Features

- 🎤 Voice recording of memories
- 📝 Automatic speech-to-text transcription
- 🔍 Semantic search across memories
- 😊 Emotion analysis from conversations
- 📅 Timeline view of saved memories
- 🤖 AI-powered memory assistant
- 🔒 Privacy-focused local storage

## Tech Stack

1. Android Frontend

- Kotlin
- Android Studio
- Retrofit
- SQLite
- Material Design

2. Backend

- Python
- Flask
- REST APIs

3. AI / Machine Learning

- Whisper (Speech-to-Text)
- Sentence Transformers
- FAISS (Semantic Search)
- Hugging Face Transformers
- Emotion Detection

## Getting Started

I. Android App

1. Open `android-app` in Android Studio.
2. Sync Gradle.
3. Run the application on an emulator or Android device.

II. Backend

```bash
cd backend/timerewind-server
pip install -r requirements.txt
python app.py
```
